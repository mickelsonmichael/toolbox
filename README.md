# Mike's Toolbox

Mike's Toolbox is a static, browser-only collection of utility tools.

## Included tools
- Base64 Encoder/Decoder
- JSON/YAML Formatter (pretty-format only)
- URL Encoder/Decoder

## Run
No build or server is required. Open `index.html` directly in a browser.

## Notes
- UI uses Bootstrap via CDN.
- Icons use Font Awesome Free via CDN.
- Shared page shell is provided by the `toolbox-layout` Web Component (`assets/layout.js`).
- Formatter prettifying uses Prettier via CDN.
- Each tool uses its own folder with `index.html`, `styles.css`, and `script.js`.
- Updates are tracked in `changelog.html`.
